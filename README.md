**3D First-Person Shooter (FPS) / Tower Defense Prototype**

Arcano-Tek Core is an action and strategy game prototype developed in JavaScript using the **Three.js** (WebGL) library. Your goal is to protect the central **Core** from waves of enemies. To do this, you must move around the battlefield, shoot them with your weapons, and build automated towers.
